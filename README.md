# Soaply 🧼📺

**Soaply** არის პლატფორმა სერიალების მოყვარულთათვის. აქ შეგიძლიათ აწარმოოთ თქვენი ნანახი სერიალების დღიური, შექმნათ ვოჩლისტები და სხვა პერსონალური სიები.  დაამატოთ მეგობრები, გაუგზავნოთ მათ რეკომენდაციები და თვალი ადევნოთ მათ აქტივობებს, როგორიცაა სხვადასხვა სერიალებზე ან კონკრეტულ ეპიზოდებზე დაწერილი review-ები და 5 ვარსკვლავიანი სისტემით შეფასებები. 

---

##  როგორ გავუშვათ პროექტი:
1. დარწმუნდით, რომ თქვენს კომპიუტერზე ჩართულია **Docker Desktop**.
2. გახსენით ტერმინალი პროექტის მთავარ საქაღალდეში და ააწყვეთ იმიჯი:
გაუშვით ბრძანება ტერმინალში: docker compose up --build
3. შესაძლოა ბექენდის ლინკზე მოგიწიოთ continiue to localhost unsafe-ზე დაჭერით ნებართვის მიცემა (გადადით მაგალითად ამ ლინკზე: https://localhost:8443/api/shows/trending და აქედან მიეცით ნებართვა)
4. შემდეგ დოკერიდანვე გადადით frontend-ის ლინკზე. (port: 5173)
